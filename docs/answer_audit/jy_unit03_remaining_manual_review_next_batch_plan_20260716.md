## 四、截圖品質要求

1. 使用 `input/JY-人身保險.pdf`，不要使用 OCR 純文字或 output JSON 畫面。
2. 截圖需原尺寸或足以清楚辨識中文字、標點、A–D 標記、組合選項與右側答案數字。
3. 不得只截答案欄；每題必須同時看到題號、完整題幹、全部選項與答案。
4. 若頁面為雙欄排版，需保留欄線或足夠邊界，避免把相鄰題目的文字誤歸入本題。
5. 若答案欄與題目不在同一張圖，可補一張重疊截圖，重疊範圍需包含題號以利對應。

## 五、收到截圖後的處理順序

1. 逐張確認頁碼、題號與版面邊界。
2. 逐題抄錄原稿題幹、選項、答案及可見解析。
3. 比對既有文字盤點與正式題庫搜尋結果。
4. 只有原稿證據完整的題目，才另案建立漏題候選確認文件並更新 correction ledger。
5. 即使確認原稿內容，仍維持 `pending_inclusion_review`；未建立補題規則前不得新增到正式題庫。
