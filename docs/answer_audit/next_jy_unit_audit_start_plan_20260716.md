## 五、PDF 截圖需求

### 啟動階段

目前**不需要使用者立即提供 PDF 截圖**。本地已有：

- `input/JY-人身保險.pdf`
- `shiwu_raw.txt`
- `output/JY-人身保險.json`
- `all_questions.json`

可先完成 P.93 Q1–Q14 的唯讀來源清單、候選匹配與污染盤點。

### 後續可能需要截圖的情況

若唯讀盤點後仍有下列情況，才請使用者補充原稿截圖：

1. raw text 與 output JSON 都無法可靠還原題幹、選項或答案欄。
2. 頁面欄位順序造成答案無法對應題號。
3. 題幹或解析跨頁，無法確認文字歸屬。
4. 正反問法、組合選項或教材註解需要視覺版面確認。

屆時截圖應以 P.93 為第一優先，逐段包含題號、完整題幹、所有選項、右側答案欄及可見解析；本計畫階段不預先要求截圖。
