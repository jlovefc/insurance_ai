# 14 題漏題候選等價題檢查審核決策

## 一、文件目的與適用範圍

本文件依據下列文件及使用者人工審核裁定，記錄 `MISS-20260716-0001` 至 `MISS-20260716-0014` 共 14 題漏題候選的處理結論：

1. `docs/answer_audit/missing_question_inclusion_rules_20260716.md`
2. `docs/answer_audit/missing_question_equivalence_check_20260716.md`
3. `docs/corrections/correction_ledger_20260716.md`
4. `docs/answer_audit/jy_unit03_manual_review_closeout_20260716.md`

本文件是等價題檢查後的審核決策紀錄，不是正式補題核准或資料修改指令。所有漏題候選仍維持 `confirmed_by_source / pending_inclusion_review`，不得因本文件改列 `ready_to_fix`。本階段不建立新題 ID、不建立 ID mapping、不補入題庫，也不修改 `all_questions.json` 或 SQLite。

## 二、可進入補題計畫的題目

### 1. MISS-20260716-0004／P.90 Q19

- 裁定：可進入補題計畫。
- 題目考點：預定死亡率降低時，定期保險保費的變化。
- 理由：正式題庫未見同題或等價題。已找到的相近題分別涉及預定利率或年金保險死亡率，因素、商品及考點均不同。
- 後續限制：本裁定僅允許進入補題 ID mapping 草案規劃，不代表已核准正式補入。

### 2. MISS-20260716-0001／P.89 Q12

- 裁定：可有條件進入補題計畫。
- 題目考點：依投保人數、死亡率及保額，以收支相等概念計算每人純保費。
- 理由：本題雖與既有題使用相同純保費計算公式，但數值情境、保額、死亡率及選項均不同；此類「同公式、不同數字」可視為不同計算題並存。
- 條件：後續建立 ID mapping 前，仍須再次確認正式題庫不存在數值、保額、死亡率及選項均完全相同的版本。
- 後續限制：條件未完成前不得正式補入。

## 三、暫緩補題：需教材版本／法規時點審查

### 1. MISS-20260716-0008／P.90 Q29

- 裁定：暫緩。
- 理由：題目涉及不分紅保險單招攬廣告及預定利率規範，必須保留教材版本語境，並另行執行 `outdated_law`／教材版本審查。
- 處理原則：完成專門審查前不得進入 ID mapping 或正式補題。

### 2. MISS-20260716-0010／P.91 Q35

- 裁定：暫緩。
- 理由：題目涉及二十五年滿期生死合險修正制的制度起始年度，須進行教材版本及歷史時點審查。
- 處理原則：不得未經版本審查便以現行法或現行制度覆蓋教材答案。

### 3. MISS-20260716-0011／P.91 Q36

- 裁定：暫緩。
- 理由：題目涉及二十年繳費終身保險修正制的制度起始年度，須進行教材版本及歷史時點審查。
- 處理原則：完成審查前不得進入 ID mapping 或正式補題。

## 四、暫緩補題：需等價題人工確認

### MISS-20260716-0013／P.91 Q47

- 裁定：暫緩。
- 疑似等價既有題：ID 107、ID 3795、ID 3796。
- 理由：上述既有題與本題均涉及保單紅利支付方式，概念高度重疊；但正反問法不同，且「積存方法」與既有題用語是否等價仍需人工確認。
- 處理原則：在人工判定既有題是否已涵蓋同一教材考點、以及同概念不同問法是否允許並存前，不得直接補入。
